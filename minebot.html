Управляйте ботом из чата - Круглосуточный стрим

04.07.2020

На стриме можно отдавать команды боту в чате и он будет делать их в Майнкрафте. Можно использовать команды импакта, баритона и мои собственные.

IP Сервера: 2b2t.org.ru

Коды управления новое

Терперь можно управлять ботом единолично, без того, чтобы кто-то другой мешал. Для этого есть коды управления, которые даются на 1, 3 и 5 минут. Получить их можно будет тут.

Применить код очень просто. Надо просто написать в чате .control код и вверху экрана появится таймер с вашим ником.

Коды можно получить здесь

Перемещение:

Перемещаться можно несколькими способами.

.goto

  • .goto 999 999
    • Перемещает на 999 по x и 999 по z
  • .goto 999
    • Перемещает на 999 по y
  • .goto 999 999 999
    • Перемещает на 999 по x, 999 по z и на 999 по y

Координаты должны быть больше 500 по модулю

.goto с названием

Можно перемещаться к блокам и объектам. Названия блоков можно посмотреть здесь (например crafting_table), а список извесных мне объектов вот:

  • .goto portal
  • .goto end_portal

координаты эндер порталов за пределами радиуса 10к можно поискать тут (нужно выбрать версию Java 1.9-1.12)

.b follow

Следует за чем-то. Можно ввести например .b follow entity player или .b follow entity item или .b follow entity cow.

.fix

Если игрок застрял, введи .fix и он будет двигаться в разные стороны.

.b invert

Повернуть назад

.b cancel / .b stop / .goto cancel

Эта команда останавливает бота.

Остальное

Более полный список передвижений можно найти здесь.

Крафт

Механика крафта написана мной, поэтому команды все от меня. Чтобы открыть инвентарь, надо ввести .e. При открытом инвентаре все прочие команды, кроме .move и .slot недоступны.

Чтобы открывать верстаки, сундуки и прочее, надо воспользоваться командой .goto с названием нужного блока.

.move

Эта команда перемещает предметы по слотам. Есть три возможных функции

  • l - переместится на слот и нажать левый клик
  • r - переместится на слот и нажать правый клик
  • m - просто переместиться на слот

После функции идут номера слотов, в которые надо переместится перед тем, как нажать. Можно указать просто один номер слота, несколько номеров через запятую или последовательность через две точки (например 1..4 превратится в 1,2,3,4).

Эти функции можно соединять знаком +.

Номера слотов можно увидеть на этой картинке:

Вот несколько примеров команд:

  • .move m1
  • .move l1,28
  • .move l28+r47..50+l51+l8

.craft

Есть макросы команды .move для крафта. Пока я добавил только крафт верстака .craft crafting_table слот_с_бревнами слот_куда_положить_верстак.

.slot

Команда слот просто жмет кнопки от 1 до 9. Можно использовать также вне инвентаря.

Разные полезные телодвижения

.t hunt

C .t hunt можно охотиться на указанные объекты (игроки, животные). Настраивать это можно через .set hunt.

.b farm

С помощью .b farm можно искать и собирать урожай. Если ты видишь грядку арбузов - введи .b farm и они будут собраны.

.b mine

Ищет блоки и ломает их. Список с названиями блоков тут (например crafting_table).

.b tunnel

Эта команда копает почти прямой тонель в один блок.

.b build

Можно строить различные схематики. Например .b build crafting_table.schematic чтобы поставить верстак. Присылайте мне свои схематики, чтобы я добавил их боту в комментах

Способности

  • .t killAura
    • Бьет все, что видит
  • .t aimBot
    • Смотрит на все, что видит
  • .t jesus
    • Ходит по воде
